# Next.js Example

React Form Builder integration with Next.js for server-side rendering.

## Prerequisites

- Node.js >= 18.0.0
- React >= 18.0.0

## Installation

```bash
npm install --legacy-peer-deps
# or
yarn install
```

## Development

```bash
# Build the form builder library (from root)
npm run build

# Start Next.js dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the example.

## Features

- Server-side rendering compatible
- React 18 with createRoot API
- Form builder and generator integration

## Notes

- Ensure the main library is built before running this example
- Uses `--legacy-peer-deps` for installation due to peer dependency transitions

For more information, see the [main examples README](../README.md).
